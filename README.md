# SmartCADD: An AI-Integrated Drug Designing Platform

## Current Version: **0.1**

### Version **0.01** Notes:

-   Initial version
-   Basic data structure is pandas dataframe

## Installation

```bash
git clone git@github.com:SMU-CATCO/SmartCADD.git
cd SmartCADD

conda create -n smartcadd python=3.10
conda activate smartcadd

pip install poetry
poetry install

pip install -e
```
